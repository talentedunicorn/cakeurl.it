Cakeurl.it
==========

A URL shortener with cakePHP. 
Still in development. 

Install instructions
--------------------
* Install cakePHP v.2.3+
* Create a database (MySQL recommended)
* Import schema file to database from app/Config/Schema/db_cakeurlit.sql
* Enjoy!
